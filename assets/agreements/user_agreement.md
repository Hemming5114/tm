## 2. 服务描述

甜梦是一款旅行分享社交应用，为用户提供以下主要服务：
- 旅行内容发布与分享
- 用户间社交互动（关注、收藏、评论）
- 即时聊天功能
- 视频通话功能
- 个人资料管理
- 内容浏览与发现

## 3. 账户注册与管理

### 3.1 账户注册
- 您需要创建账户才能使用本应用的完整功能
- 您应提供准确、完整的注册信息
- 您有责任维护账户信息的准确性

### 3.2 账户安全
- 您有责任保护您的账户安全
- 请勿与他人共享您的账户信息
- 如发现账户被盗用，请立即联系我们

### 3.3 账户终止
- 您可以随时删除您的账户
- 我们保留在违反本协议情况下终止您账户的权利

## 4. 用户行为规范

### 4.1 禁止行为
在使用本应用时，您不得：
- 发布违法、有害、威胁、辱骂、诽谤、色情、淫秽或其他不当内容
- 侵犯他人的知识产权、隐私权或其他合法权益
- 传播虚假信息或误导性内容
- 进行任何形式的骚扰、欺凌或恶意行为
- 上传含有病毒、木马等恶意代码的内容
- 进行商业推广或垃圾信息传播
- 使用自动化工具或机器人程序

### 4.2 内容规范
- 您发布的所有内容应真实、合法
- 尊重他人的知识产权，不得抄袭或盗用他人作品
- 保持积极、友善的社区氛围

### 4.3 违规处理
对于违反本协议的行为，我们有权采取以下措施：
- 警告提醒
- 删除违规内容
